# RAFI — Frontend (Vercel)

> Next.js application deployed on Vercel.
> Will be initialized in Phase 2.

## Stack
- **Framework**: Next.js 15 (App Router)
- **Deployment**: Vercel
- **Wallet**: Phantom / Backpack / Solflare via `@solana/wallet-adapter`
- **State**: React Query + Anchor client
- **Styling**: CSS Modules / Vanilla CSS

# RAFI — TypeScript SDK

> Client library for interacting with the RAFI on-chain program.
> Will be built after the smart contract is stabilized.

## Planned exports
- `createPool()` — seller creates a pool with asset deposit
- `mintProbability()` — buyer acquires probability %
- `settlePool()` — trigger VRF settlement
- `claimRefund()` — buyer refund from expired pool
- `fetchPool()` — read pool state
- `fetchPositions()` — list all positions for a pool
